## Description
A daily prediction game for World ID-verified humans. One question per day, one vote per person. Vote on real-world outcomes, see community results, and climb the leaderboard.

## World ID Integration
- Orb-level verification required to participate
- Server-side proof verification via verifyCloudProof
- One-person-one-vote guaranteed (nullifier hash unique constraint)

## How it drives verified human engagement
- Daily question creates habitual return behavior (like Wordle)
- Streak system incentivizes consecutive daily participation
- Points + leaderboard competition drives engagement
- Share/deep link feature enables organic viral growth
- 3-language support (EN/JA/ES) for global reach
